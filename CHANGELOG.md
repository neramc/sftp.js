# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.2] - 2026-01-04

### Added
- Initial release
- SFTP connection management with password and private key authentication
- File upload and download functionality
- Directory upload and download (recursive)
- File and directory deletion
- Directory creation (recursive)
- File and directory listing
- File existence checking
- File rename and move operations
- File permission management (chmod)
- File content read and write operations
- Connection retry mechanism
- Comprehensive error handling
- Full API documentation

### Features
- Support for both CommonJS and ES6 module systems
- Connection state management
- Automatic retry on connection failures
- Detailed response objects with success status and metadata

## [Unreleased]

### Planned

- Connection pooling
- Stream-based file operations
